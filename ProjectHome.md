# Introduction #
寫過 code 的人都知道, 每一種語言都會提供常用的函數來滿足常用的功能. 當函數一多變成了一個 library. 這個 project 也是, 在我平時工作中累積了一些常用的函數但是在 JDK 中又沒有提供時, 就只好透過這個 project 來記錄且提供日後參考, 以避免重複的 google 與 coding.
